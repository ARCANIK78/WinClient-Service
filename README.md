# WinClient-Service

Guía de **Configuración** para el despliegue de servicios persistentes en entornos Windows utilizando **NSSM**.

## Arquitectura
Esta configuración permite integrar un stack completo para que funcione de forma automatizada:
* **Frontend:** Interfaz de usuario.
* **Backend:** API de gestión.
* **Base de Datos:** Persistencia con PostgreSQL.
